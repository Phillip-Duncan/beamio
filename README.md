
# beamio

Beamio: A python package for handling of input/output/conversion of Acquired or Modelled beam data
for use in further analysis or conversion between vendors and/or software versions.

## Features

- [x] IBA Accept (v6) ascii IO.
- [x] Raystation v2025+ csv IO.
- [x] Conversion between beam data models and IO using converted data.
